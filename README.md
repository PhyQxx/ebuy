<p align=center>
   EBuy 是一个网上商城项目
</p>

## 项目选型

采用SpringBoot + Beetl2


## 网站功能

- 采用了时下流行的SpringBoot + Beetl2技术。
- MySQL，严格的数据库规范设计。

## 环境依赖

| 环境依赖                 | 说明描述                          |
| ------------------------ | ----------------------------- |
| 软件基础环境             | 服务器（例:1 vCPU 2 GiB 1Mbps）+域名(例：kiwipeach.cn) ，注意这里的域名需要备案才能够访问喔|
| 数据库                 | MySQL |
| JDK | 推荐使用jdk1.8及1.8以上的版本 |
| node | 前端需要使用node环境启动前端服务，其中请求转发可以使用前端提供转发工具，当然也可以使用nginx，看具体部署条件。 |
